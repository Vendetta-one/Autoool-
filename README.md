# Autool-

A dedicated ride-sharing platform for the JUIT Waknaghat community.

Autool helps students connect at the main gate to pool autos and taxis, making travel to Shimla, Solan, and Chandigarh more affordable and efficient.

Features

Real-time Pools: Find students heading your way right now.

Cost Sharing: Split the fare 3 ways and save money.

Verified Community: Exclusive to JUIT students for better safety.

Built with ❤️ for the JUIT community.
